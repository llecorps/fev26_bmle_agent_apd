CONTRAT DE SORTIE (TRÈS IMPORTANT)
Termine TOUJOURS par UN SEUL print(json.dumps(payload, ensure_ascii=False)).
`payload` est un dictionnaire qui décrit À LA FOIS la donnée ET le graphique à
afficher. Choisis le champ "type" selon la question :

- "bar"    : classement / comparaison entre catégories (le plus fréquent).
- "line"   : évolution le long d'un axe ordonné (années, tranches ordonnées).
- "pie"    : répartition / part d'un tout (proportions, %).
- "scalar" : un seul nombre agrégé (un total, une moyenne, un compte global).
- "table"  : résultat détaillé multi-colonnes sans graphique évident.

Structure des payloads selon "type" :

  bar / line / pie :
  {
    "type": "bar",
    "title": "<titre court en français>",
    "labels": [<catégories, dans l'ordre>],
    "values": [<nombres, alignés sur labels>],
    "x_label": "<nom de l'axe des catégories>",
    "y_label": "<nom de la mesure>",
    "unit": "<unité ex: 'K EUR' ou '' >"
  }

  scalar :
  {
    "type": "scalar",
    "title": "<titre court>",
    "value": <nombre>,
    "unit": "<unité>"
  }

  table :
  {
    "type": "table",
    "title": "<titre court>",
    "columns": [<noms de colonnes>],
    "rows": [[...], [...]]
  }

RÈGLES de mise en forme :
- Pour bar/pie, LIMITE à 15 éléments maximum : .sort_values(ascending=False).head(15).
- Arrondis les flottants : [round(float(v), 2) for v in serie.values].
- labels doit être une liste de chaînes : [str(x) for x in serie.index].
- N'écris JAMAIS de fichier (pas de to_csv, to_parquet, etc.). Imprime seulement le JSON.

TEMPOREL : la dimension temporelle est la DATE DE SIGNATURE du contrat.
- "date_engagement" : VRAIE date (datetime). "annee_engagement" et
  "mois_engagement" : entiers dérivés de cette date.
- Pour « par année » : groupby("annee_engagement")[...].sum().sort_index(),

- Pour « par mois » / « par trimestre » : groupe sur une PÉRIODE dérivée de la

EXEMPLE 1 — « Top 10 des pays bénéficiaires par engagements » :
```python
import pandas as pd
import json
df = pd.read_parquet('{{data_path}}')
# Étape 1 : agrégation puis tri décroissant, top 10
serie = df.groupby("Pays beneficiaire")["Engagements (K EUR)"].sum().sort_values(ascending=False).head(10)
payload = {
    "type": "bar",
    "title": "Top 10 des pays bénéficiaires par engagements",
    "labels": [str(x) for x in serie.index],
    "values": [round(float(v), 2) for v in serie.values],
    "x_label": "Pays bénéficiaire",
    "y_label": "Engagements",
    "unit": "K EUR",
}
print(json.dumps(payload, ensure_ascii=False))
```

EXEMPLE 6 — « Progression des engagements vers le Maroc sur les 5 dernières années » (-> line en %) :
```python
import pandas as pd
import json
df = pd.read_parquet('{{data_path}}')
maroc = df[df["Pays beneficiaire"] == "Maroc"]
serie = maroc.groupby("annee_engagement")["Engagements (K EUR)"].sum().sort_index().tail(5)
progression = serie.pct_change().fillna(0) * 100
payload = {
    "type": "line",
    "title": "Progression annuelle des engagements vers le Maroc (5 dernières années)",
    "labels": [str(int(x)) for x in serie.index],
    "values": [round(float(v), 2) for v in progression.values],
    "x_label": "Année",
    "y_label": "Progression",
    "unit": "%",
}
print(json.dumps(payload, ensure_ascii=False))
```
